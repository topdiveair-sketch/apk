Zuhause am Bach Mobile Android V1.0

Dieses ZIP enthält ein Android-Studio-Projekt.

Funktionen:
- 4 Zimmer: DZ Bachblick, Marillenzimmer, Weinbergzimmer, Donauzimmer
- Gäste speichern / neu hinzufügen
- Buchungen auf verschiedene Zimmer
- Doppelbelegungsschutz
- Extras/Getränke/Speisen zur Rechnung buchen
- Große Bedienfelder
- Offline-Speicherung am Handy

APK erstellen:
1. Android Studio installieren.
2. Diesen Projektordner öffnen.
3. Gradle Sync abwarten.
4. Build > Build Bundle(s) / APK(s) > Build APK(s).
5. APK: app/build/outputs/apk/debug/app-debug.apk

Installation am Handy:
APK aufs Handy kopieren, antippen, Installation erlauben.
